# DESCRIPTION OF TASK

A Java based console application that consumes real-time weather data from a public REST API (OpenWeatherMap) and displays it in a structured format. The program should make HTTP requests, parse JSON responses, and present key weather  details such as temperature, humidity, and weather conditions.


*Task Requirements*
1. Fetch Weather Data:
   > Use the OpenWeatherMap API to get weather data for a speciified city.
   > Construct a proper API URL with an API key and city name.

2. Handle HTTP Request:
   > Utilize Java's HttpURLConnection or HttpClient to send GET requests.
   > Handle network errrors and invalid responses gracefully.

3. Parses JSON Responses:
   > Use a JSON parsing library (e.g., org.json) to extract relevant weather details.
   > Displays temperature, humidity, and weather descriptions in a user-friendly format.


*Tools and Technologies*
> Java
> JSON Library (org.json)
> OpenWeatherMap API
